## Project Description
This project is a Spring Boot MVC application that demonstrates the use of controllers, request mapping, and Thymeleaf templates. The application is accessed through a web browser and follows the MVC pattern.

## Technologies Used
- Java
- Spring Boot
- Spring Web
- Thymeleaf

## How to Run the Project
1. Open the project in IntelliJ IDEA
2. Run the application using Task1Application
3. Application starts at: http://localhost:8080
